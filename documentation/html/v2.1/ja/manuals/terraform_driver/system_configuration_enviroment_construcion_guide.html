システム構成/環境構築ガイド Terraform-driver編¶

はじめに¶

本書では、ITAでTerraformオプション機能(以下、Terraform driver)として運用する為のシステム構成と環境構築について説明します。
ITA Terraform driverを利用するにあたっては、ITA基本機能が構築済みであることが前提です。ITA基本機能の構築に関しては、「環境構築ガイド(基本編)」をご覧ください。
